# 项目技术栈总结

## 项目概述
这是一个基于 Vue.js 的博客系统前端项目（demo-web），采用现代化的前端技术栈构建。

## 核心技术栈

### 1. 前端框架
- **Vue.js 2.6.14**
  - 渐进式 JavaScript 框架
  - 采用组件化开发模式
  - 使用 Vue CLI 5.0 作为构建工具

### 2. UI 框架
- **Vuetify 2.6.15**
  - Material Design 组件库
  - 提供丰富的 UI 组件（卡片、对话框、表单等）
  - 响应式设计支持

### 3. 状态管理
- **Vuex 3.6.2**
  - 集中式状态管理
  - 管理用户信息、网站配置、搜索状态等
- **vuex-persistedstate 4.1.0**
  - Vuex 状态持久化插件
  - 将状态保存到 localStorage

### 4. 路由管理
- **Vue Router 3.6.5**
  - 单页面应用路由管理
  - 支持路由守卫、动态路由等

### 5. HTTP 客户端
- **Axios 1.3.6**
  - 基于 Promise 的 HTTP 库
  - 支持请求/响应拦截器
  - 统一处理 API 请求和错误

### 6. 样式处理
- **SCSS/SASS**
  - CSS 预处理器
  - 支持变量、嵌套、混入等特性
  - 使用 `node-sass 8.0.0` 和 `sass-loader 13.2.2`

### 7. 构建工具
- **Vue CLI 5.0**
  - 基于 Webpack 的构建工具
  - 支持热重载、代码分割等
  - 配置了路径别名（@、assets、components、utils）

### 8. 代码转换
- **Babel**
  - ES6+ 代码转换
  - 使用 `@babel/preset-env` 预设
  - 配置了 `@babel/core` 和 `@babel/eslint-parser`

## 功能库

### 1. Markdown 编辑器
- **@kangc/v-md-editor 1.7.12**
  - Vue Markdown 编辑器
  - 支持编辑和预览模式
  - 用于文章编辑功能

### 2. 动画库
- **animate.css 3.7.2**
  - CSS 动画库
  - 提供丰富的动画效果
- **wowjs 1.1.3**
  - 滚动动画库
  - 实现页面滚动时的动画效果

### 3. 无限滚动
- **vue-infinite-loading 2.4.5**
  - Vue 无限滚动加载组件
  - 用于文章列表的懒加载

### 4. 弹幕效果
- **vue-baberrage 3.2.4**
  - Vue 弹幕组件
  - 用于评论或消息展示

### 5. 工具库
- **js-cookie 3.0.5**
  - Cookie 操作库
  - 用于用户认证信息存储
- **core-js 3.8.3**
  - JavaScript 标准库 polyfill
  - 提供 ES6+ 特性支持

## 开发工具

### 1. 代码质量
- **ESLint 7.32.0**
  - JavaScript 代码检查工具
  - 配置了 `eslint-plugin-vue` 插件
  - 使用 `@babel/eslint-parser` 作为解析器

### 2. 测试框架
- **Jest 26.6.3**
  - JavaScript 测试框架
  - 支持单元测试、快照测试
  - 配置了代码覆盖率收集
- **@vue/test-utils 1.3.0**
  - Vue 组件测试工具库
  - 提供 `shallowMount`、`mount` 等方法
- **vue-jest 4.0.1**
  - Vue 单文件组件测试转换器
- **babel-jest 26.6.3**
  - Babel 转换器用于 Jest
- **jsdom 16.7.0**
  - DOM 环境模拟
  - 用于测试环境中的 DOM 操作

### 3. 其他工具
- **svg-sprite-loader 6.0.11**
  - SVG 图标处理
  - 将 SVG 文件转换为 sprite
- **node-polyfill-webpack-plugin 2.0.1**
  - Node.js polyfill 插件
  - 为浏览器环境提供 Node.js API

## 项目结构

```
demo-web/
├── src/
│   ├── api/              # API 接口定义
│   ├── assets/           # 静态资源（字体、图片、配置）
│   ├── components/       # 公共组件
│   │   ├── banner/       # 轮播图组件
│   │   ├── comment/       # 评论组件
│   │   ├── layout/       # 布局组件（Header、Footer、Sidebar）
│   │   ├── model/        # 模态框组件（登录、注册等）
│   │   ├── pagination/   # 分页组件
│   │   └── ...
│   ├── icons/            # SVG 图标
│   ├── plugins/          # 插件配置（Vuetify）
│   ├── router/           # 路由配置
│   ├── store/            # Vuex 状态管理
│   ├── tests/            # 测试文件
│   │   ├── mocks/        # Mock 文件
│   │   └── unit/         # 单元测试
│   ├── utils/            # 工具函数
│   │   ├── auth.js       # 认证相关
│   │   ├── request.js    # HTTP 请求封装
│   │   └── ...
│   └── view/             # 页面组件
│       ├── article/      # 文章相关页面
│       ├── home/         # 首页
│       ├── user/         # 用户中心
│       └── ...
├── public/               # 公共静态文件
├── babel.config.js       # Babel 配置
├── jest.config.js        # Jest 测试配置
├── vue.config.js         # Vue CLI 配置
└── package.json          # 项目依赖配置
```

## 浏览器支持

根据 `browserslist` 配置：
- 支持市场份额 > 1% 的浏览器
- 支持最近 2 个版本
- 排除已停止维护的浏览器

## 开发环境

- **Node.js**: 建议使用 LTS 版本
- **包管理器**: npm
- **开发服务器**: 运行在端口 80
- **构建工具**: Vue CLI Service

## 运行命令

```bash
# 开发环境
npm run serve

# 生产构建
npm run build

# 代码检查
npm run lint

# 运行所有测试
npm run test:all

# 运行单个测试
npm run test:searchindex

# 生成测试覆盖率
npm run test:coverage:all
```

## 总结

这是一个现代化的 Vue.js 博客前端项目，采用了：
- **Vue 2.x** 作为核心框架
- **Vuetify** 提供 Material Design UI
- **Vuex** 进行状态管理
- **Axios** 处理 HTTP 请求
- **Jest** 进行单元测试

- 丰富的第三方库支持各种功能需求

项目结构清晰，代码组织良好，具有良好的可维护性和可扩展性。
